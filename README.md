# Haramain Guide

A full-featured Hajj & Umrah web application.

## Features
- Live Makkah & Madinah
- Qibla direction
- Tasbi counter
- Historical places
- Hotels & Hospitals
- Admin panel (CRUD)
- Multi-language (BN, EN, AR)

## Tech Stack
- React + Vite
- Node.js + Express
- SQLite
- Tailwind CSS

## Run Project

### Backend
cd server  
npm install  
npm run dev  

### Frontend
cd client  
npm install  
npm run dev
